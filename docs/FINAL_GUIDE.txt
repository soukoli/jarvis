  Cmd+; = ⌘+;

  On Mac keyboard:
    ⌘ (Command) + ; (Semicolon)

  Location:
    Command = Left of spacebar
    ; = Right side, next to 'L' key

  Why Cmd+;?
    ✓ Easy one-hand press
    ✓ Doesn't conflict with VS Code
    ✓ Doesn't conflict with browsers
    ✓ Semicolon rarely used alone
    ✓ Fast to trigger
